# Jenkins-Setup-on-EC2

## 📌 Overview
Setup and configured Jenkins on an EC2 instance for CI/CD automation.

## ⚙️ Steps
1. Launch EC2 instance
2. Install Java
3. Install Jenkins
4. Configured Jenkins dashboard
5. Configure pipelines

## 🚀 Features
- Automated builds
- Continuous integration
- Pipeline configuration
